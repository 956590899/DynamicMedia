# 灵动岛 (MediaIsland)

一个美观、实用的媒体信息显示工具，支持灵动岛模式和悬浮歌词功能。

## 功能特性

- 🎵 **灵动岛模式**：在屏幕顶部显示媒体信息，支持动画效果
- 📝 **悬浮歌词**：在屏幕底部显示实时歌词，支持动画效果
- 🎨 **音频可视化**：显示音频频谱，支持多种可视化效果
- 🔄 **自动更新**：实时获取并更新媒体信息
- 🎯 **响应式设计**：自适应不同屏幕分辨率
- 📱 **交互功能**：支持专辑图点击操作

## 安装方法

### 1. 安装依赖

项目需要以下依赖包：
- pillow
- winsdk
- pycaw
- PyQt5
- numpy
- sounddevice
- requests
- mutagen

可以使用以下命令安装：

```bash
pip install -r requirements.txt
```

或者运行项目目录下的 `依赖.bat` 脚本自动安装。

### 2. 运行项目

直接运行 `dynamic.py` 文件即可启动项目：

```bash
python dynamic.py
```

## 操作逻辑

### 基本操作

| 操作 | 功能 |
|------|------|
| **左单击专辑图** | 取消退出倒计时 / 临时隐藏并清除缓存 |
| **右双击专辑图** | 触发退出倒计时 |
| **双击歌曲名/歌手名** | 切换悬浮歌词显示状态 |
| **点击音频可视化区域** | 切换音频可视化样式 |

### 退出逻辑

1. **触发退出**：右双击专辑图
2. **隐藏悬浮歌词**：自动隐藏现有悬浮歌词
3. **显示倒计时**：显示读秒悬浮窗口，倒计时5秒
4. **取消退出**：在倒计时期间左单击专辑图
5. **确认退出**：倒计时结束后自动退出程序

### 悬浮歌词操作

1. **显示/隐藏**：双击歌曲名或歌手名
2. **颜色切换**：右键点击悬浮歌词，选择颜色
3. **拖拽移动**：在非背景穿透模式下可拖拽移动

## 配置说明

项目的主要配置参数位于 `dynamic.py` 文件的开头部分：

### 悬浮歌词配置

- `FLOAT_LYRIC_ENABLED`：悬浮歌词开关（0=关闭，1=开启）
- `FLOAT_LYRIC_TEXT_COLOR`：歌词颜色
- `FLOAT_LYRIC_TEXT_SIZE`：歌词字体大小
- `FLOAT_LYRIC_BACKGROUND_OPACITY`：背景透明度
- `FLOAT_LYRIC_BOTTOM_MARGIN`：屏幕下方间距

### 灵动岛配置

- `DYNAMIC_ISLAND_ENABLED`：灵动岛模式开关（0=关闭，1=开启）
- `DYNAMIC_ISLAND_OPACITY`：灵动岛透明度

### 音频可视化配置

- `VISUALIZATION_STYLE`：可视化样式（3=圆形频谱，4=波浪形，5=关闭）

## 支持的格式

### 音频格式

- MP3（支持ID3标签）
- FLAC
- OGG
- MP4

### 歌词格式

- 内嵌歌词（支持USLT、COMM标签）
- 本地LRC文件
- 在线歌词（通过网络搜索）

## 常见问题

### 1. 悬浮歌词不显示

- 检查 `FLOAT_LYRIC_ENABLED` 是否设置为 1
- 确保音乐正在播放且有歌词
- 检查是否有内嵌歌词或本地LRC文件

### 2. 音频可视化不显示

- 检查 `VISUALIZATION_STYLE` 是否设置为 3 或 4
- 确保音乐正在播放
- 检查是否有音频输出

### 3. 灵动岛不显示

- 检查 `DYNAMIC_ISLAND_ENABLED` 是否设置为 1
- 确保音乐正在播放
- 检查是否有媒体会话
